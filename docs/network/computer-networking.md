## HTTPS 为什么非对称加密比对称加密效率低
非对称加密算法涉及的数学操作（例如大数因子分解、离散对数问题等）相对复杂，这增加了计算的时间和资源消耗。  

而对称加密主要的运算是位运算，速度非常快，如果使用硬件计算，速度会更快。
